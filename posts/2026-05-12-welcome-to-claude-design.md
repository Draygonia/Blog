# Welcome to Claude Design

This is the beginning of something. A space to collect ideas, share inspiration, and think through design.

## What to expect

- **Posts** — thoughts on design, process, and creative work
- **Links** — a curated collection of resources, tools, and inspiration worth bookmarking

## How it works

Everything here is stored directly in a GitHub repository as plain text files — no database, no CMS. Posts are Markdown files, links are a JSON file.

To add a post or link, head to the [Admin panel](/admin.html), sign in with a GitHub Personal Access Token, and publish. Changes save instantly to the repo.

## Getting started

1. Go to [Admin &rarr; admin.html](/admin.html)
2. Generate a [GitHub token](https://github.com/settings/tokens/new?scopes=repo) with `repo` scope
3. Sign in and start writing
